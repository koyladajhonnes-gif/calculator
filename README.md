# calculator
A simple Command-Line Calculator built in Python. Supports basic arithmetic operations like addition, subtraction, multiplication, and division. Designed for beginners to practice Python functions, loops, and user input handling.
Overview
This is a simple Command-Line Calculator built using Python.  
It performs basic arithmetic operations and runs directly in the terminal.
Features
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Continuous loop until the user exits
- Handles invalid inputs safely
Technologies Used
-Python 3
-VS Code / Terminal
